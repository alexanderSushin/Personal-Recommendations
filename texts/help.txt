/random - рекомендация случайного аниме
/top_alltime - топ аниме по рейтингу за все время
/top_month - топ аниме по рейтингу за месяц
/top_week - топ аниме по рейтингу за неделю
/personal - персональная рекомендация аниме
/add_friend $username$ - отправить запрос добавления в друзья пользователю с никнеймом $username$ (вводить $username$, нужно без знака $)
/remove_friend $username$ - удалить из друзей пользователя с никнеймом $username$ (вводить $username$, нужно без знака $)
/all_friends - список всех ваших друзей
/top_friend $username$ - топ аниме по рейтингу у $username$ (вводить username, нужно без знака $)
/top_all_friends - топ аниме по рейтингу у всех ваших друзей
/announce - анонс ближайших новый серий аниме
/subscribe $name$ - подписаться на уведомления о анонсе новых серий аниме с названием $name$ (вводить $name$, нужно без знака $, вводить $name$ можно не полностью, бот поймет по примерному названию корректное)
/unsubscribe $name$ - отписаться на уведомления о анонсе новых серий аниме с названием $name$ (вводить $name$, нужно без знака $, вводить $name$ можно не полностью, бот поймет по примерному названию корректное)
/rate $name$ - указать рейтинг просмотренного аниме (вводить $name$, нужно без знака $, вводить $name$ можно не полностью, бот поймет по примерному названию корректное)
